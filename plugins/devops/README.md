# 🎉 devops

Plugin for **DevOps workflow** — manages session context and streamlines working with Claude Code.

### ⭐ Skills

| Skill | วัตถุประสงค์ |
|---|---|
| `session-name` | ตั้งชื่อ session ปัจจุบัน บันทึกลง memory และแสดง context สำหรับอ้างอิงในการสนทนา |

### 🏆 Usage

```
/session-name <ชื่อ session>
```
