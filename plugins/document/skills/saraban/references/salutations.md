# คำขึ้นต้น สรรพนาม และคำลงท้าย ตามฐานะผู้รับ
## (ภาคผนวก ๒ ของระเบียบสำนักนายกรัฐมนตรีว่าด้วยงานสารบรรณ)

reference นี้สำคัญมาก — คำขึ้นต้น (เรียน/กราบเรียน/นมัสการ/กราบทูล) และคำลงท้าย **เปลี่ยนตามฐานะผู้รับ**
ถ้าใช้ผิดจะกลายเป็นเอกสารที่ไม่ถูกต้องทันที การใช้ "เรียน...ขอแสดงความนับถือ" ใช้ได้เฉพาะ
**บุคคลทั่วไป/ข้าราชการทั่วไป** เท่านั้น — ผู้รับระดับสูงหรือพระภิกษุใช้สูตรต่างออกไป

> **ก่อนร่างหนังสือทุกครั้ง ให้ระบุก่อนว่าผู้รับอยู่กลุ่มไหน** แล้วเลือกคำขึ้นต้น-ลงท้ายจากตารางด้านล่าง

---

## กลุ่ม ก — บุคคลทั่วไป (ค่าเริ่มต้น)

| รายการ | คำที่ใช้ |
|--------|---------|
| คำขึ้นต้น | เรียน [ตำแหน่ง/ชื่อ] |
| สรรพนามแทนผู้รับ | ท่าน |
| สรรพนามแทนผู้เขียน | ข้าพเจ้า / กระผม (ชาย) / ดิฉัน (หญิง) / [ชื่อหน่วยงาน] |
| คำลงท้าย | ขอแสดงความนับถือ |
| คำจ่าหน้าซอง | เรียน |

ใช้กับ: ข้าราชการทั่วไป ผู้บริหารหน่วยงาน เอกชน ประชาชนทั่วไป และกรณีส่วนใหญ่

---

## กลุ่ม ข — บุคคลระดับสูง (ใช้ "กราบเรียน")

| รายการ | คำที่ใช้ |
|--------|---------|
| คำขึ้นต้น | กราบเรียน [ตำแหน่ง] |
| สรรพนามแทนผู้รับ | ท่าน |
| สรรพนามแทนผู้เขียน | ข้าพเจ้า / กระผม / ดิฉัน |
| คำลงท้าย | ขอแสดงความนับถืออย่างยิ่ง |
| คำจ่าหน้าซอง | กราบเรียน |

ใช้กับตำแหน่งเหล่านี้เท่านั้น:
- ประธานองคมนตรี
- นายกรัฐมนตรี
- ประธานรัฐสภา / ประธานสภาผู้แทนราษฎร / ประธานวุฒิสภา
- ประธานศาลฎีกา / ประธานศาลรัฐธรรมนูญ / ประธานศาลปกครองสูงสุด
- ประธานกรรมการการเลือกตั้ง / ประธานกรรมการสิทธิมนุษยชนแห่งชาติ
- ประธานกรรมการ ป.ป.ช. / ประธานกรรมการตรวจเงินแผ่นดิน
- ผู้ตรวจการแผ่นดิน / อัยการสูงสุด
- รัฐบุรุษ

---

## กลุ่ม ค — พระภิกษุ

คำลงท้าย "ขอนมัสการด้วยความเคารพ" มี **3 ระดับ** ตามสมณศักดิ์ — ระวังอย่าใช้ผิดระดับ

| ผู้รับ | คำขึ้นต้น | สรรพนามแทนผู้รับ | คำลงท้าย |
|--------|----------|-----------------|----------|
| สมเด็จพระราชาคณะ / รองสมเด็จพระราชาคณะ | นมัสการ | พระคุณเจ้า | ขอนมัสการด้วยความเคารพอย่างยิ่ง |
| พระราชาคณะ (ชั้นราช เทพ ธรรม พรหม) | นมัสการ | พระคุณท่าน | ขอนมัสการด้วยความเคารพอย่างสูง |
| พระภิกษุทั่วไป (พระครูสัญญาบัตร/พระเปรียญ/พระทั่วไป) | นมัสการ | ท่าน | ขอนมัสการด้วยความเคารพ |

- สรรพนามแทนผู้เขียน: กระผม / ดิฉัน (พระราชาคณะขึ้นไป) หรือ ผม / ดิฉัน (พระทั่วไป)
- คำจ่าหน้าซอง: นมัสการ

---

## กลุ่ม ง — สมเด็จพระสังฆราช และพระราชวงศ์ (ราชาศัพท์ชั้นสูง)

กลุ่มนี้ใช้ราชาศัพท์ระดับสูงและพบไม่บ่อย **ต้องตรวจสอบความถูกต้องกับสำนักงานราชบัณฑิตยสภาหรือสำนักพระราชวังสำหรับแต่ละพระองค์เสมอ** — เพราะคำเปลี่ยนตามพระอิสริยยศเฉพาะบุคคล ห้ามเดารูปแบบ

**สมเด็จพระสังฆราช** (ยืนยันจากแหล่งราชการ):
| รายการ | คำที่ใช้ |
|--------|---------|
| คำขึ้นต้น | กราบทูล [พระนาม] |
| สรรพนามแทนผู้รับ | ใต้ฝ่าพระบาท |
| สรรพนามแทนผู้เขียน | เกล้ากระหม่อม (ชาย) / เกล้ากระหม่อมฉัน (หญิง) |
| คำลงท้าย | ควรมิควรแล้วแต่จะโปรด |

**พระราชวงศ์** (สมเด็จเจ้าฟ้า / พระองค์เจ้า / หม่อมเจ้า) และ **พระบาทสมเด็จพระเจ้าอยู่หัว / สมเด็จพระบรมราชินี**:
ใช้ราชาศัพท์เฉพาะ (กราบบังคมทูล / ขอเดชะ / ควรมิควรแล้วแต่จะทรงพระกรุณาโปรดเกล้าฯ ฯลฯ)
→ **ให้ค้นคู่มือราชาศัพท์ของราชบัณฑิตยสภาก่อนร่างทุกครั้ง** อย่าใช้จากความจำ

---

## หลักปฏิบัติ
- ถ้าไม่แน่ใจว่าผู้รับอยู่กลุ่มไหน โดยเฉพาะกลุ่ม ค และ ง → **ถามผู้ใช้หรือค้นหายืนยันก่อน** อย่าเดา
- คำขึ้นต้นในหัวหนังสือ กับคำที่ใช้จ่าหน้าซอง ต้องเป็นคำเดียวกัน (เรียน→เรียน, กราบเรียน→กราบเรียน)
- คำลงท้ายต้องสอดคล้องกับคำขึ้นต้นเสมอ (กราบเรียน คู่กับ ขอแสดงความนับถืออย่างยิ่ง ไม่ใช่ ขอแสดงความนับถือ)

*อ้างอิง: ภาคผนวก ๒ ระเบียบสำนักนายกรัฐมนตรีว่าด้วยงานสารบรรณ พ.ศ. ๒๕๒๖ — ดู `references/regulations.md`*
