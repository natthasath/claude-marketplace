# รูปแบบเอกสารและงานเพิ่มเติม

reference นี้เสริมส่วนที่ `document-structure.md` และ `letter-templates.md` ยังไม่ครอบคลุม:
หนังสือประทับตรา, หนังสือเวียน, การรับรองสำเนาถูกต้อง และการจ่าหน้าซอง

---

## 1. หนังสือประทับตรา

หนังสือที่ใช้ **ตราส่วนราชการประทับแทนการลงลายมือชื่อ** ของหัวหน้าส่วนราชการระดับกรมขึ้นไป
โดยหัวหน้าส่วนราชการระดับกอง หรือผู้ที่ได้รับมอบหมาย เป็นผู้ลงลายมือชื่อย่อกำกับตรา
ใช้ได้ทั้งระหว่างส่วนราชการด้วยกัน และระหว่างส่วนราชการกับบุคคลภายนอก
**เฉพาะกรณีที่ไม่ใช่เรื่องสำคัญ** เช่น การขอรายละเอียดเพิ่มเติม การส่งสำเนาเอกสาร
การแจ้งเตือนเรื่องที่ค้าง การตอบรับทราบทั่วไป

### องค์ประกอบ
```
[ตราครุฑ]

ที่ [รหัส]/[เลขที่]

ถึง [ชื่อส่วนราชการผู้รับ]          ← ใช้ "ถึง" ไม่ใช่ "เรียน"

    [เนื้อหา]

                    [ชื่อส่วนราชการเจ้าของหนังสือ]
                    (ประทับตราส่วนราชการ + ลงชื่อย่อกำกับตรา)
                    วันที่ [วันที่เต็ม]

[ส่วนราชการเจ้าของเรื่อง]
โทร. [เบอร์]
```

### กฎเฉพาะ
- ขึ้นต้นด้วย "ถึง" (เหมือนบันทึกข้อความ แต่ใช้ระหว่างหน่วยงาน)
- ลงท้ายด้วยชื่อส่วนราชการ + ประทับตรา + ลายมือชื่อย่อกำกับตรา (ไม่มีลายมือชื่อเต็มของหัวหน้าส่วนราชการ ไม่มี "ขอแสดงความนับถือ")
- ต้องระบุวันเดือนปีที่ออกหนังสือ

---

## 2. หนังสือเวียน

หนังสือที่ส่งถึงผู้รับ**จำนวนมากพร้อมกัน** โดยมีข้อความเดียวกัน (เช่น แจ้งเวียนระเบียบใหม่)

### การกำหนดเลข "ว"
- ให้มีตัวอักษร **"ว"** นำหน้าเลขทะเบียนหนังสือส่ง
- รูปแบบ: `ที่ [รหัสหน่วยงาน]/ว [เลข]`
- ตัวอย่าง: `ที่ ศธ ๐๕๐๑/ว ๑๒๓`

### ลักษณะ
- โครงสร้างเนื้อหาเหมือนหนังสือภายนอกหรือบันทึกข้อความปกติ
- เปลี่ยนเฉพาะการกำหนดเลข "ว" และช่อง "เรียน" ที่อาจใช้ "เรียน หัวหน้าส่วนราชการทุกส่วน" หรือระบุกลุ่มผู้รับ

---

## 3. การรับรองสำเนาถูกต้อง

เมื่อทำสำเนาเอกสารราชการเพื่อใช้แทนต้นฉบับ ต้องรับรองสำเนาถูกต้องที่มุมล่างของสำเนา:

```
                        สำเนาถูกต้อง

                        (ลงลายมือชื่อ)
                    [ชื่อ-นามสกุลตัวบรรจง]
                        [ตำแหน่ง]
                        [วัน เดือน ปี]
```

### กฎเฉพาะ
- เขียนคำว่า "สำเนาถูกต้อง" แล้วลงลายมือชื่อผู้รับรอง
- เขียนชื่อเต็มตัวบรรจงกำกับใต้ลายมือชื่อ พร้อมตำแหน่ง
- ผู้รับรองควรเป็นเจ้าหน้าที่ที่เกี่ยวข้องกับเอกสารนั้น

---

## 4. การจ่าหน้าซอง

| ตำแหน่งบนซอง | ข้อมูล |
|-------------|--------|
| มุมบนซ้าย | ชื่อและที่อยู่ส่วนราชการผู้ส่ง + เลขที่หนังสือ |
| กลางซอง | คำขึ้นต้น + ชื่อ/ตำแหน่งผู้รับ + ที่อยู่ |
| มุมบน (ถ้ามี) | ชั้นความเร็ว/ชั้นความลับ ด้วยตัวอักษรสีแดง |

**คำขึ้นต้นบนซองต้องตรงกับในหนังสือ:**
- บุคคลทั่วไป → "เรียน"
- ระดับสูง → "กราบเรียน"
- พระภิกษุ → "นมัสการ"
