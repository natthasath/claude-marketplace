# คู่มือภาษาราชการ — ภาษาพูด vs. ภาษาราชการ

---

## คำสรรพนาม

| บริบท | ภาษาพูด/ทั่วไป | ภาษาราชการ | หมายเหตุ |
|-------|--------------|-----------|---------|
| ผู้เขียน (ชาย) | ผม | กระผม / ข้าพเจ้า | กระผม = ชายเท่านั้น |
| ผู้เขียน (หญิง) | ฉัน / หนู | ดิฉัน / ข้าพเจ้า | |
| ผู้เขียน (ทั่วไป) | เรา | ข้าพเจ้า / [ชื่อหน่วยงาน] | หน่วยงานอ้างตัวด้วยชื่อ |
| ผู้รับ | คุณ | ท่าน | |
| บุคคลที่ 3 | เขา / เธอ / มัน | ท่าน / ผู้นั้น / [ชื่อ/ตำแหน่ง] | |
| หน่วยงานตัวเอง | บริษัท/หน่วยงาน | [ชื่อหน่วยงานเต็ม] | ห้ามใช้สรรพนามแทนหน่วยงาน |

**ตัวอย่าง:**
- ✗ `เราต้องการขอให้คุณช่วย...`
- ✓ `[ชื่อหน่วยงาน] ขอความอนุเคราะห์จากท่าน...`

## คำทดแทน — ภาษาพูด → ภาษาราชการ

### คำกริยาและกริยาวลี

| ภาษาพูด | ภาษาราชการ |
|---------|-----------|
| บอก | แจ้ง / เรียน / แจ้งให้ทราบ |
| ส่ง | จัดส่ง / นำส่ง / ส่งมอบ |
| ขอให้ / กรุณา | โปรด / ขอได้โปรด |
| ทำ | ดำเนินการ / จัดทำ / ดำเนินงาน |
| ทำเรื่อง | ดำเนินการ / ยื่นเรื่อง |
| เช็ค / ตรวจ | ตรวจสอบ / ตรวจสอบข้อเท็จจริง |
| ดู | พิจารณา / ตรวจสอบ |
| คุย | หารือ / ประสานงาน |
| บอกต่อ | แจ้งเวียน / ประสานงาน |
| รับทราบ | รับทราบ (ใช้ได้) / รับทราบแล้ว |
| โอเค / เห็นด้วย | เห็นชอบ / อนุมัติ / เห็นควร |
| ไม่โอเค / ปฏิเสธ | ไม่เห็นชอบ / พิจารณาแล้วเห็นว่าไม่เหมาะสม |
| ยืนยัน | ยืนยัน / รับรอง |
| อัพเดต / แจ้งความคืบหน้า | รายงานความคืบหน้า |
| ติดตาม | ติดตามผล / เร่งรัด |
| ช่วย | ให้ความอนุเคราะห์ / ให้ความร่วมมือ |
| เรียกประชุม | นัดประชุม / กำหนดประชุม |
| ขอโทษ | ขออภัย |
| ขอบคุณ | ขอบคุณ (ใช้ได้) / ขอบพระคุณ (ทางการมากขึ้น) |

### คำนามและวลี

| ภาษาพูด | ภาษาราชการ |
|---------|-----------|
| คนงาน / พนักงาน | เจ้าหน้าที่ / บุคลากร |
| หัวหน้า | ผู้บังคับบัญชา / ผู้บริหาร |
| ลูกน้อง | ผู้ใต้บังคับบัญชา |
| ปัญหา | ปัญหา / อุปสรรค / ข้อขัดข้อง |
| ผล | ผลการดำเนินการ / ผลการพิจารณา |
| เรื่อง | เรื่อง / กรณีดังกล่าว / ในการนี้ |
| เงิน | งบประมาณ / เงินทุน / ค่าใช้จ่าย |
| เดือนที่แล้ว | ห้วงเดือน[ชื่อเดือน] ที่ผ่านมา |
| เร็วๆ นี้ | ในเร็วนี้ / โดยเร็ว / ในโอกาสอันสมควร |
| ทันที | โดยด่วน / โดยเร็วที่สุด |
| ตอนนี้ / ปัจจุบัน | ปัจจุบัน / ณ บัดนี้ |
| ต่อไป | ต่อไปนี้ / ต่อไปในอนาคต |
| ครั้งนี้ | ในคราวนี้ / คราวนี้ |
| รายละเอียด | รายละเอียด / รายการ |
| ตกลง / ข้อตกลง | ข้อตกลง / บันทึกข้อตกลง (MOU) |
| เหตุผล | เหตุผล / เหตุอันสมควร |

### คำเชื่อมและสำนวน

| ภาษาพูด | ภาษาราชการ |
|---------|-----------|
| เพราะ | เนื่องจาก / เนื่องด้วย |
| เพื่อ | เพื่อ / เพื่อประโยชน์ |
| แต่ว่า | ทั้งนี้ / อย่างไรก็ตาม / อย่างไรก็ดี |
| ดังนั้น | ดังนั้น / ด้วยเหตุนี้ / ในการนี้ |
| ถ้า | หาก / กรณีที่ |
| ก็ | จึง |
| อีกทั้ง | นอกจากนี้ / ยิ่งไปกว่านั้น |
| ด้วย (ท้ายประโยค) | ด้วย (ใช้ต้นประโยคเท่านั้น) |

## วันที่และตัวเลข

### รูปแบบวันที่
| บริบท | รูปแบบ |
|-------|-------|
| หนังสือภายนอก | วันที่ ๑ มกราคม พ.ศ. ๒๕๖๘ |
| บันทึกข้อความ | ๑ มกราคม ๒๕๖๘ (ยืดหยุ่นกว่า) |
| ในเนื้อหา | วันที่ [วันที่] เดือน[เดือน] พ.ศ. [ปี] |

**ห้ามใช้:**
- ตัวเลขย่อ: `1/1/68`, `01-01-2568`
- ย่อเดือน: `ม.ค.`, `Feb.`
- คริสตศักราช (ค.ศ.) เว้นแต่เป็นเอกสารระหว่างประเทศ

### ตัวเลขในเนื้อหา
| กรณี | รูปแบบ |
|------|-------|
| ในเนื้อหาทั่วไป | ตัวเลขอารบิก (1, 2, 3) ยอมรับได้ |
| ลำดับข้อ | ตัวเลขไทย (๑, ๒, ๓) |
| จำนวนเงินในสัญญา | ตัวอักษรตามด้วยตัวเลข: "สองแสนบาทถ้วน (200,000 บาท)" |
| ปีพุทธศักราช | พ.ศ. เสมอ |

## ภาษาอังกฤษในเอกสารราชการ

### หลักการ
1. **หลีกเลี่ยง** การใช้คำภาษาอังกฤษปนในประโยคภาษาไทย
2. **ถ้าหลีกเลี่ยงไม่ได้** ให้ใส่คำแปลภาษาไทยก่อน แล้วตามด้วยวงเล็บ

> **หมายเหตุ:** หลักข้อนี้ว่าด้วย *คำศัพท์ในเนื้อหา* เท่านั้น ไม่เกี่ยวกับ *ช่องทางส่ง* —
> การส่งหนังสือราชการทางไปรษณีย์อิเล็กทรอนิกส์ (อีเมล) เป็นช่องทางที่ชอบด้วยระเบียบแล้ว
> ตามฉบับที่ ๔ พ.ศ. ๒๕๖๔ (ดู `references/electronic-saraban.md`)

| ภาษาพูด | ภาษาราชการ |
|---------|-----------|
| Email / อีเมล | ไปรษณีย์อิเล็กทรอนิกส์ (E-mail) |
| Online | ออนไลน์ / ผ่านระบบอินเทอร์เน็ต |
| Website | เว็บไซต์ / เว็บไซต์ (Website) |
| Update | ปรับปรุง / ปรับปรุงข้อมูล |
| Meeting | การประชุม |
| Report | รายงาน |
| Budget | งบประมาณ |
| Project | โครงการ |
| Software | ซอฟต์แวร์ / โปรแกรมคอมพิวเตอร์ |
| Hardware | ฮาร์ดแวร์ / อุปกรณ์คอมพิวเตอร์ |
| AI | ปัญญาประดิษฐ์ (AI) |
| Database | ฐานข้อมูล |

## ข้อผิดพลาดที่พบบ่อย

### 1. ใช้ภาษาพูดขึ้นต้นหนังสือ
- ✗ `ผมต้องการแจ้งให้ทราบว่า...`
- ✓ `ด้วย[หน่วยงาน] ขอแจ้งให้ทราบว่า...`

### 2. ลงท้ายผิดสูตร
- ✗ `หวังว่าจะได้รับความร่วมมือ`
- ✓ `จึงเรียนมาเพื่อโปรดพิจารณาให้ความอนุเคราะห์ด้วย จะขอบคุณยิ่ง`

### 3. ขาดประธานในประโยค
- ✗ `ขอแจ้งให้ทราบว่า...` (ไม่รู้ว่าใครแจ้ง)
- ✓ `[ชื่อหน่วยงาน] ขอแจ้งให้ทราบว่า...`

### 4. ใช้คำสั่งตรงๆ โดยไม่มี "โปรด"
- ✗ `กรุณาส่งเอกสารภายในวันที่ 30`
- ✓ `โปรดส่งเอกสารภายในวันที่ ๓๐ [เดือน] พ.ศ. ๒๕๖๘`

### 5. อ้างถึงเอกสารไม่ถูกรูปแบบ
- ✗ `ตามที่ได้คุยกันไว้`
- ✓ `ตามที่ได้หารือร่วมกัน ในการประชุมวันที่ ๑ มกราคม พ.ศ. ๒๕๖๘`
- ✓ `อ้างถึง หนังสือ[หน่วยงาน] ที่ กค ๐๔๐๑/๑๒๓ ลงวันที่ ๑ มกราคม พ.ศ. ๒๕๖๘`

### 6. ใช้เครื่องหมายไม่เหมาะสม
- ✗ ใช้ `!`, `?`, `...` ในเนื้อหาหนังสือราชการ
- ✓ ไม่ใช้เครื่องหมายปฏิทัน ใช้ประโยคที่ชัดเจนแทน
- ✗ `โปรดส่งภายในพรุ่งนี้!!!`
- ✓ `โปรดส่งภายในวันที่ ๑๕ มกราคม พ.ศ. ๒๕๖๘ โดยด่วน`

### 7. ย่อหน้าไม่ถูกต้อง
- เนื้อหาต้องขึ้นต้นด้วยการย่อหน้า (Tab หรือ 2 เคาะ)
- แต่ละย่อหน้าต้องมีความคิดหลักเดียว
- ประโยคที่ขึ้นต้นด้วย "ด้วย / ตามที่ / ตามสิ่งอ้างถึง" ต้องย่อหน้าเสมอ
