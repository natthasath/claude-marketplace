# โครงสร้างหนังสือราชการ
## ตามระเบียบสำนักนายกรัฐมนตรีว่าด้วยงานสารบรรณ พ.ศ. ๒๕๒๖ และแก้ไขเพิ่มเติม พ.ศ. ๒๕๔๘

---

## ประเภทหนังสือราชการ (6 ประเภท)

| ประเภท | ใช้เมื่อ | ระดับทางการ |
|--------|---------|-------------|
| หนังสือภายนอก | ติดต่อระหว่างหน่วยงาน หรือบุคคลภายนอก | สูงสุด |
| หนังสือภายใน (บันทึกข้อความ) | ติดต่อภายในหน่วยงานเดียวกัน | กลาง |
| หนังสือประทับตรา | แจ้งเวียนทั่วไป ไม่ต้องลายเซ็น | กลาง |
| หนังสือสั่งการ (คำสั่ง/ระเบียบ/ข้อบังคับ) | ออกคำสั่งหรือกำหนดระเบียบ | สูง |
| หนังสือประชาสัมพันธ์ (ประกาศ/แจ้งความ/แถลงการณ์) | เผยแพร่ข้อมูลแก่สาธารณะ | กลาง-สูง |
| หนังสือที่เจ้าหน้าที่ทำขึ้น (รายงาน/บันทึก) | เอกสารภายในสำหรับบันทึก | กลาง |

## 1. หนังสือภายนอก

### องค์ประกอบ (เรียงตามตำแหน่งจากบนลงล่าง)

```
[ตราครุฑ]                    ← บนสุด ตรงกลาง (ขนาด 3 ซม.)

ที่ [รหัสหน่วยงาน]/[เลขที่]     วันที่ ๑ มกราคม พ.ศ. ๒๕๖๘
                                 ← ชิดขวา
เรื่อง [หัวเรื่อง]

เรียน [ชื่อตำแหน่งหรือชื่อบุคคล]

[อ้างถึง หนังสือ... ลงวันที่...]     ← มีเฉพาะกรณีตอบหรืออ้างอิง
[สิ่งที่ส่งมาด้วย ๑. ... ๒. ...]     ← มีเฉพาะกรณีมีเอกสารแนบ

    [เนื้อหา — ขึ้นต้นด้วยย่อหน้า]
    [วรรคแรก: ที่มา/บริบท]
    [วรรคกลาง: รายละเอียด/ความต้องการ]
    [วรรคสุดท้าย: ลงท้ายด้วยสูตรราชการ]

                    ขอแสดงความนับถือ   ← ชิดขวา

                    (ลายเซ็น)
                [ชื่อ-นามสกุล (ตัวพิมพ์)]
                    [ตำแหน่ง]

[ชื่อส่วนราชการ]
โทร. XXXX
โทรสาร. XXXX
```

### กฎเฉพาะ
- **ตราครุฑ**: หน่วยงานราชการใช้ครุฑ / รัฐวิสาหกิจใช้ตราสัญลักษณ์ของตน
- **ที่**: ใช้รหัสหน่วยงานนำหน้า เช่น `กค ๐๔๐๑/๑๒๓`
- **วันที่**: เขียนเต็มเสมอ ห้ามย่อ → `๑ ม.ค. ๖๘` ใช้ไม่ได้
- **เรื่อง**: กระชับ ไม่เกิน 1 บรรทัด ไม่ใส่เครื่องหมายจุดท้าย
- **เรียน**: ใส่ชื่อตำแหน่ง ไม่ใส่คำนำหน้า เช่น "เรียน ผู้อำนวยการสำนักงาน..." (ไม่ใช่ "คุณ...")

## 2. บันทึกข้อความ (หนังสือภายใน)

### องค์ประกอบ

```
บันทึกข้อความ                ← หัวเรื่อง ตรงกลาง

ส่วนราชการ [ชื่อฝ่าย/กอง]  ที่ [เลขที่]  วันที่ [วันที่เต็ม]

เรื่อง [หัวเรื่อง]

ถึง [ชื่อตำแหน่งผู้รับ]

    [เนื้อหา]

                (ลายเซ็น)
            [ชื่อ-นามสกุล]
                [ตำแหน่ง]
```

### กฎเฉพาะ
- ใช้ "ถึง" แทน "เรียน"
- ไม่ต้องมีตราครุฑ แต่บางหน่วยงานนิยมใส่
- ลงท้ายไม่ต้องมี "ขอแสดงความนับถือ"
- ภาษายืดหยุ่นกว่าหนังสือภายนอก แต่ยังต้องเป็นทางการ

### ข้อยกเว้นสำหรับหน่วยงานในกำกับของรัฐ
หน่วยงานที่เป็น **สถาบันอุดมศึกษาในกำกับของรัฐ** (เช่น NIDA, มหาวิทยาลัยในกำกับอื่นๆ)
อาจมีระเบียบงานสารบรรณของตนเองที่กำหนดให้ใช้คำว่า **"ส่วนงาน"** แทน **"ส่วนราชการ"**
ในหัวบันทึกข้อความ ตามศัพท์ทางกฎหมายเฉพาะของสถาบัน (พ.ร.บ. จัดตั้งของแต่ละแห่ง)

ก่อน flag คำว่า "ส่วนงาน" ว่าผิด ให้ตรวจสอบก่อนว่าผู้ใช้สังกัดหน่วยงานประเภทใด —
ถ้าเป็นหน่วยงานในกำกับที่มีระเบียบเฉพาะ ให้ยึดตามระเบียบของหน่วยงานนั้นเป็นหลัก
ไม่ใช่ template มาตรฐานของระเบียบสำนักนายกฯ พ.ศ. ๒๕๒๖

## 3. ประกาศ

### องค์ประกอบ

```
ประกาศ[ชื่อหน่วยงาน]
เรื่อง [หัวเรื่อง]
─────────────────────

    [เนื้อหา — ขึ้นต้นด้วย "โดยที่..." หรือ "ตามที่..."]
    [ข้อ ๑ ... ข้อ ๒ ... (กรณีมีข้อกำหนดหลายข้อ)]

    ประกาศ ณ วันที่ [วันที่เต็ม]

                (ลายเซ็น)
            [ชื่อ-นามสกุล]
        [ตำแหน่งผู้ออกประกาศ]
```

## 4. คำสั่ง

### องค์ประกอบ

```
คำสั่ง[ชื่อหน่วยงาน]
ที่ [เลขที่]/[ปีพ.ศ.]
เรื่อง [หัวเรื่อง]
─────────────────────

    [เนื้อหา — มักขึ้นต้นด้วย "โดยที่..." หรือ "ด้วย..."]
    [ข้อ ๑ ... ข้อ ๒ ...]
    [ข้อสุดท้าย: "ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป" หรือ "ตั้งแต่วันที่ ... เป็นต้นไป"]

    สั่ง ณ วันที่ [วันที่เต็ม]

                (ลายเซ็น)
            [ชื่อ-นามสกุล]
        [ตำแหน่งผู้ออกคำสั่ง]
```

## 5. หนังสือรับรอง

### องค์ประกอบ

```
หนังสือรับรอง

    [ชื่อหน่วยงาน] ขอรับรองว่า [ชื่อบุคคล] [รายละเอียดที่รับรอง]

    หนังสือรับรองฉบับนี้ออกให้เพื่อ[วัตถุประสงค์] เท่านั้น

    ออกให้ ณ วันที่ [วันที่เต็ม]

                (ลายเซ็น)
            [ชื่อ-นามสกุล]
                [ตำแหน่ง]
```
