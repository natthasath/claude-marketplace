# การรับ-ส่ง และการเก็บรักษาหนังสือราชการ

งานทะเบียนรับ-ส่งเป็นงานประจำวันหลักของเจ้าหน้าที่สารบรรณ
reference นี้ครอบคลุมขั้นตอนการลงรับ การส่งออก และการเก็บรักษา-ทำลายหนังสือ

---

## การรับหนังสือ (หนังสือเข้า)

เมื่อหนังสือมาถึงหน่วยงาน ให้ดำเนินการตามลำดับ:

1. **จัดลำดับความสำคัญ** — แยกหนังสือด่วน/ลับ ออกก่อน
2. **ประทับตรารับหนังสือ** ที่มุมบนขวาของหนังสือ ตรารับประกอบด้วย:
   ```
   ┌─────────────────────────┐
   │   [ชื่อส่วนราชการ]        │
   │  เลขรับ ............     │
   │  วันที่ ............     │
   │  เวลา  ............     │
   └─────────────────────────┘
   ```
3. **ลงทะเบียนรับ** ในทะเบียนหนังสือรับ (สมุดหรือระบบ e-Saraban) บันทึก:
   - เลขทะเบียนรับ (เรียงลำดับต่อเนื่องตลอดปี)
   - ที่ของหนังสือ / ลงวันที่ / จาก (หน่วยงานผู้ส่ง) / ถึง / เรื่อง
   - การปฏิบัติ / หมายเหตุ
4. **ส่งให้ผู้เกี่ยวข้อง** ดำเนินการต่อ

---

## การส่งหนังสือ (หนังสือออก)

1. **ลงเลขที่หนังสือส่ง** — "ที่" ของหนังสือ ใช้รหัสหน่วยงาน + เลขทะเบียนส่ง
   เช่น `ที่ ศธ ๐๕๐๑/๑๒๓`
2. **ลงทะเบียนหนังสือส่ง** บันทึก: เลขทะเบียนส่ง / ที่ / ลงวันที่ / ถึง / เรื่อง
3. **สำเนาเก็บเข้าแฟ้ม** อย่างน้อย ๑ ฉบับ
4. **นำส่ง** ตามช่องทาง (ไปรษณีย์ / นำส่งเอง / ไปรษณีย์อิเล็กทรอนิกส์ตาม `references/electronic-saraban.md`)

**หมายเหตุเรื่องเลขที่หนังสือ:**
- หนังสือทั่วไป: `ที่ [รหัส]/[เลขทะเบียน]`
- หนังสือเวียน: มีอักษร "ว" นำหน้าเลข → `ที่ [รหัส]/ว [เลข]` (ดู `references/additional-forms.md`)

---

## การเก็บรักษาและอายุหนังสือ

| ระยะ | การปฏิบัติ |
|------|-----------|
| ระหว่างปฏิบัติ | เก็บในแฟ้มที่ค้นหาได้ง่าย |
| เก็บเมื่อปฏิบัติเสร็จ | ตามกำหนดเวลาในระเบียบ (หมวด ๓ ข้อ ๕๗) — โดยทั่วไปหนังสือที่เก็บโดยมีกำหนดเวลา เก็บไม่น้อยกว่า ๑๐ ปี เว้นแต่กำหนดเป็นอย่างอื่น |
| หนังสือที่เก็บถาวร | เอกสารสำคัญ (เช่น ประวัติศาสตร์ สถิติ ต้นเรื่องกฎหมาย) ส่งมอบหอจดหมายเหตุแห่งชาติ |

**การทำลายหนังสือ (หมวด ๓ ข้อ ๖๖–๗๐):**
- ภายใน ๖๐ วันหลังสิ้นปีปฏิทิน ให้เจ้าหน้าที่สำรวจหนังสือที่ครบอายุการเก็บ
- จัดทำบัญชีหนังสือขอทำลาย เสนอหัวหน้าส่วนราชการแต่งตั้ง **คณะกรรมการทำลายหนังสือ**
  (ประธาน ๑ คน และกรรมการอย่างน้อย ๒ คน)
- คณะกรรมการพิจารณาแล้วจึงทำลาย — ห้ามทำลายเองโดยพลการ

---

## หลักปฏิบัติ
- เลขทะเบียนรับและเลขทะเบียนส่ง เป็นคนละชุดกัน และเรียงต่อเนื่องแยกกันตลอดปีปฏิทิน
- เมื่อผู้ใช้ถามเรื่อง "ลงรับหนังสือ" "เลขหนังสือส่ง" "ทะเบียนหนังสือ" → ใช้ reference นี้
- ปัจจุบันงานทะเบียนส่วนใหญ่ทำผ่านระบบ e-Saraban แต่หลักการลำดับเลข/บันทึกข้อมูลยังเหมือนเดิม

*อ้างอิง: ระเบียบสำนักนายกรัฐมนตรีว่าด้วยงานสารบรรณ พ.ศ. ๒๕๒๖ หมวดการรับ-ส่งและการเก็บรักษา — ดู `references/regulations.md`*
