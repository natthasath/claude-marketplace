# Template หนังสือราชการ

ใช้ template เหล่านี้เป็นโครงสร้างเริ่มต้น ปรับเนื้อหาตามสถานการณ์จริง
เครื่องหมาย `[...]` = ข้อมูลที่ต้องเติม | `(ถ้ามี)` = ส่วนที่ไม่บังคับ

---

## 1. หนังสือภายนอก — แจ้งให้ทราบ

```
ที่ [รหัส]/[เลขที่]

                                        วันที่ [วันที่เต็ม]

เรื่อง [หัวเรื่องกระชับ]

เรียน [ตำแหน่งผู้รับ]

    ด้วย[หน่วยงานผู้ส่ง]ได้[ดำเนินการ/พิจารณา/ดำเนินโครงการ]
[รายละเอียดที่มาและบริบท]

    ในการนี้ [หน่วยงาน/ข้าพเจ้า] จึงขอแจ้งให้ท่านทราบว่า
[สาระสำคัญที่ต้องการแจ้ง]

    จึงเรียนมาเพื่อโปรดทราบ

                                        ขอแสดงความนับถือ

                                        (ลายเซ็น)
                                    [ชื่อ-นามสกุล]
                                        [ตำแหน่ง]

[ส่วนราชการ/ฝ่าย]
โทร. [เบอร์]
```

---

## 2. หนังสือภายนอก — ขอความอนุเคราะห์/ขอความร่วมมือ

```
ที่ [รหัส]/[เลขที่]

                                        วันที่ [วันที่เต็ม]

เรื่อง ขอความอนุเคราะห์[สิ่งที่ต้องการ]

เรียน [ตำแหน่งผู้รับ]

    ด้วย[หน่วยงาน]มีความประสงค์จะ[วัตถุประสงค์]
เนื่องจาก[เหตุผล/ความจำเป็น]

    จึงใคร่ขอความอนุเคราะห์จากท่านได้โปรด[สิ่งที่ขอ]
ภายในวันที่ [กำหนดเส้นตาย]  (ถ้ามี)

    จึงเรียนมาเพื่อโปรดพิจารณาให้ความอนุเคราะห์ด้วย จะขอบคุณยิ่ง

                                        ขอแสดงความนับถือ

                                        (ลายเซ็น)
                                    [ชื่อ-นามสกุล]
                                        [ตำแหน่ง]

[ส่วนราชการ/ฝ่าย]
โทร. [เบอร์]
```

---

## 3. หนังสือภายนอก — ตอบรับ/ตอบกลับ

```
ที่ [รหัส]/[เลขที่]

                                        วันที่ [วันที่เต็ม]

เรื่อง [หัวเรื่อง]

เรียน [ตำแหน่งผู้รับ]

อ้างถึง หนังสือ[หน่วยงานผู้ส่งก่อน] ที่ [ที่ของหนังสือก่อน] ลงวันที่ [วันที่ของหนังสือก่อน]
(สิ่งที่ส่งมาด้วย [รายการ])  (ถ้ามี)

    ตามที่ท่านได้มีหนังสืออ้างถึง[เรื่องที่อ้าง] นั้น

    [หน่วยงาน]ขอเรียนว่า[เนื้อหาคำตอบ]

    จึงเรียนมาเพื่อโปรดทราบ

                                        ขอแสดงความนับถือ

                                        (ลายเซ็น)
                                    [ชื่อ-นามสกุล]
                                        [ตำแหน่ง]
```

---

## 4. หนังสือภายนอก — ส่งเอกสาร/รายงาน

```
ที่ [รหัส]/[เลขที่]

                                        วันที่ [วันที่เต็ม]

เรื่อง ส่ง[ชื่อเอกสาร]

เรียน [ตำแหน่งผู้รับ]

สิ่งที่ส่งมาด้วย [ชื่อเอกสาร] จำนวน [จำนวน] ฉบับ

    ตามที่[เหตุผล/คำร้องขอก่อนหน้า] นั้น

    บัดนี้[หน่วยงาน]ได้จัดทำ[ชื่อเอกสาร]เสร็จเรียบร้อยแล้ว
จึงได้จัดส่งมาพร้อมหนังสือนี้

    จึงเรียนมาเพื่อโปรดทราบ

                                        ขอแสดงความนับถือ
```

---

## 5. บันทึกข้อความ — รายงาน/แจ้งผลการดำเนินงาน

```
บันทึกข้อความ

ส่วนราชการ [ฝ่าย/กอง]  ที่ [เลขที่]  วันที่ [วันที่เต็ม]

เรื่อง รายงาน[หัวเรื่อง]

ถึง [ตำแหน่งผู้รับ]

    ตามที่[ท่าน/มีคำสั่ง/ได้มอบหมาย]ให้[หน่วยงาน/ข้าพเจ้า]
[ดำเนินการ/จัดทำ/ตรวจสอบ] นั้น

    บัดนี้[การดำเนินการ]แล้วเสร็จ ขอรายงานผลดังนี้
    ๑. [ผลดำเนินการข้อแรก]
    ๒. [ผลดำเนินการข้อสอง]
    ๓. [ข้อสังเกต/ปัญหา/ข้อเสนอแนะ (ถ้ามี)]

    จึงเรียนมาเพื่อโปรดทราบ

                                        (ลายเซ็น)
                                    [ชื่อ-นามสกุล]
                                        [ตำแหน่ง]
```

---

## 6. บันทึกข้อความ — ขออนุมัติ

```
บันทึกข้อความ

ส่วนราชการ [ฝ่าย/กอง]  ที่ [เลขที่]  วันที่ [วันที่เต็ม]

เรื่อง ขออนุมัติ[สิ่งที่ขอ]

ถึง [ตำแหน่งผู้มีอำนาจอนุมัติ]

    ด้วย[หน่วยงาน/ข้าพเจ้า]มีความจำเป็นต้อง[เหตุผล]
เพื่อ[วัตถุประสงค์]

    จึงใคร่ขออนุมัติ[สิ่งที่ขอ] เป็นจำนวน[จำนวนและหน่วย (ถ้ามี)]
งบประมาณที่ใช้ในการดำเนินการ [จำนวนเงิน] บาท (ถ้ามี)
จากงบประมาณ[หมวด/โครงการ] (ถ้ามี)

    จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติ

                                        (ลายเซ็น)
                                    [ชื่อ-นามสกุล]
                                        [ตำแหน่ง]
```

## 7. ประกาศ — ทั่วไป

```
ประกาศ[ชื่อหน่วยงาน]
เรื่อง [หัวเรื่อง]
────────────────────────────────

    โดยที่[เหตุผล/ความเป็นมา]

    อาศัยอำนาจตามความใน[กฎหมาย/ระเบียบที่ให้อำนาจ] จึงประกาศ ดังนี้

    ข้อ ๑ [ข้อกำหนดแรก]
    ข้อ ๒ [ข้อกำหนดที่สอง]
    ...
    ข้อ [สุดท้าย] ประกาศนี้ให้ใช้บังคับตั้งแต่วันที่ [วันที่] เป็นต้นไป

    ประกาศ ณ วันที่ [วันที่เต็ม]

                                        (ลายเซ็น)
                                    [ชื่อ-นามสกุล]
                                [ตำแหน่งผู้ออกประกาศ]
```

## 8. คำสั่ง — แต่งตั้ง/มอบหมาย

```
คำสั่ง[ชื่อหน่วยงาน]
ที่ [เลขที่]/[ปีพ.ศ.]
เรื่อง [แต่งตั้ง/มอบหมาย][หัวเรื่อง]
────────────────────────────────

    ด้วย[หน่วยงาน]มีความจำเป็นต้อง[เหตุผล]

    อาศัยอำนาจตามความใน[กฎหมาย/ระเบียบ] จึงมีคำสั่งดังนี้

    ข้อ ๑ ให้[ชื่อ-นามสกุล] ตำแหน่ง[ตำแหน่ง] ทำหน้าที่[หน้าที่ที่มอบหมาย]
    ข้อ ๒ ให้ผู้ที่ได้รับแต่งตั้งปฏิบัติหน้าที่ด้วยความซื่อสัตย์สุจริต
    ข้อ ๓ ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป

    สั่ง ณ วันที่ [วันที่เต็ม]

                                        (ลายเซ็น)
                                    [ชื่อ-นามสกุล]
                                [ตำแหน่งผู้ออกคำสั่ง]
```

---

## 9. หนังสือรับรอง — รับรองการทำงาน

```
หนังสือรับรอง

    [ชื่อหน่วยงาน] ขอรับรองว่า [ชื่อ-นามสกุล] อายุ [อายุ] ปี
เลขประจำตัวประชาชน [เลขบัตร] ได้ปฏิบัติงานในตำแหน่ง [ตำแหน่ง]
ของ[หน่วยงาน] ตั้งแต่วันที่ [วันที่เริ่มงาน] จนถึงปัจจุบัน
มีความประพฤติเรียบร้อย ปฏิบัติหน้าที่ด้วยความซื่อสัตย์สุจริต

    หนังสือรับรองฉบับนี้ออกให้เพื่อ[วัตถุประสงค์] เท่านั้น

    ออกให้ ณ วันที่ [วันที่เต็ม]

                                        (ลายเซ็น)
                                    [ชื่อ-นามสกุล]
                                        [ตำแหน่ง]
                                    [ชื่อหน่วยงาน]
```

## ตารางเลือกสูตรลงท้ายที่เหมาะสม

| วัตถุประสงค์ | สูตรลงท้ายที่ถูกต้อง |
|-------------|---------------------|
| แจ้งให้ทราบ | จึงเรียนมาเพื่อโปรดทราบ |
| ขอความอนุเคราะห์ | จึงเรียนมาเพื่อโปรดพิจารณาให้ความอนุเคราะห์ด้วย จะขอบคุณยิ่ง |
| ขออนุมัติ | จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติ |
| ขอความร่วมมือ | จึงเรียนมาเพื่อโปรดให้ความร่วมมือด้วย จะขอบคุณยิ่ง |
| รายงาน | จึงเรียนมาเพื่อโปรดทราบ |
| ส่งเอกสาร | จึงเรียนมาเพื่อโปรดทราบ |
| ตอบรับ | จึงเรียนมาเพื่อโปรดทราบ |
| ดำเนินการ/สั่งการ | จึงเรียนมาเพื่อโปรดพิจารณาดำเนินการต่อไปด้วย จะขอบคุณยิ่ง |
| ขอให้พิจารณา | จึงเรียนมาเพื่อโปรดพิจารณา |
