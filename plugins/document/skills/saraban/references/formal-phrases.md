# สูตรประโยคภาษาราชการ

ใช้ reference นี้เพื่อเลือกสูตรที่เหมาะสมกับวัตถุประสงค์และประเภทเอกสาร

---

## สูตรขึ้นต้นเนื้อหา (Opening Formulas)

### 1. ด้วย — กรณีแจ้งเรื่องใหม่ ไม่มีหนังสือก่อนหน้า
```
ด้วย[หน่วยงาน] ได้[ดำเนินการ/พิจารณา/ตรวจสอบ]...
ด้วย[หน่วยงาน] มีความประสงค์จะ...
ด้วย[หน่วยงาน] ขอแจ้งให้ทราบว่า...
```

### 2. ตามที่ — กรณีอ้างถึงเหตุการณ์หรือข้อเท็จจริงก่อนหน้า
```
ตามที่[หน่วยงาน/ท่าน]ได้[ดำเนินการ/แจ้ง/ร้องขอ]...
ตามที่ได้มีการประชุมร่วมกัน เมื่อวันที่...
ตามที่ปรากฏว่า...
```

### 3. ตามสิ่งอ้างถึง — กรณีมีหนังสือก่อนหน้าที่ระบุไว้ในหัว
```
ตามสิ่งอ้างถึง [หน่วยงาน] ได้มีหนังสือ[เรื่อง]...
ตามหนังสือที่อ้างถึง [หน่วยงาน] ขอเรียนว่า...
```

### 4. ตามที่ท่านได้มีหนังสือ — กรณีตอบหนังสือที่ระบุไว้ในหัว
```
ตามที่ท่านได้มีหนังสือ ที่ [ที่หนังสือ] ลงวันที่ [วันที่] เรื่อง [เรื่อง] นั้น
[หน่วยงาน] ขอเรียนว่า...
```

### 5. อนุสนธิ — กรณีเพิ่มเติมจากหนังสือก่อนหน้า (ทางการมาก)
```
อนุสนธิหนังสือ[หน่วยงาน] ที่ [ที่] ลงวันที่ [วันที่] นั้น
```

---

## สูตรลงท้ายเนื้อหา (Closing Formulas)

เลือกตามวัตถุประสงค์หลักของหนังสือ:

| วัตถุประสงค์ | สูตรลงท้าย |
|-------------|-----------|
| แจ้งให้ทราบทั่วไป | `จึงเรียนมาเพื่อโปรดทราบ` |
| แจ้งและขอให้ดำเนินการ | `จึงเรียนมาเพื่อโปรดทราบ และดำเนินการต่อไปด้วย` |
| ขออนุมัติ/ขออนุญาต | `จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติ` |
| ขอความอนุเคราะห์ | `จึงเรียนมาเพื่อโปรดพิจารณาให้ความอนุเคราะห์ด้วย จะขอบคุณยิ่ง` |
| ขอความร่วมมือ | `จึงเรียนมาเพื่อโปรดให้ความร่วมมือด้วย จะขอบคุณยิ่ง` |
| ขอให้พิจารณา | `จึงเรียนมาเพื่อโปรดพิจารณา` |
| ขอให้สั่งการ | `จึงเรียนมาเพื่อโปรดพิจารณาสั่งการต่อไปด้วย` |
| รายงานผล | `จึงเรียนมาเพื่อโปรดทราบ` |
| ส่งเอกสาร | `จึงเรียนมาเพื่อโปรดทราบ` |
| ขอเชิญประชุม | `จึงเรียนมาเพื่อโปรดเข้าร่วมประชุม ตามวัน เวลา สถานที่ดังกล่าว` |
| ตอบรับการประชุม | `จึงเรียนมาเพื่อโปรดทราบ` |
| ขอขยายเวลา | `จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติด้วย จะขอบคุณยิ่ง` |

---

## วลีกลาง — เชื่อมย่อหน้าและเนื้อหา

### เชื่อมเหตุผลและความต้องการ
```
ในการนี้ [หน่วยงาน] จึง...
เพื่อให้การดำเนินการเป็นไปด้วยความเรียบร้อย จึง...
เพื่อประโยชน์ [ของผู้รับบริการ/ของทางราชการ] จึง...
บัดนี้ [การดำเนินการ]แล้วเสร็จ จึงขอ...
```

### เชื่อมรายละเอียดเพิ่มเติม
```
นอกจากนี้ [หน่วยงาน] ขอแจ้งให้ทราบเพิ่มเติมว่า...
ทั้งนี้ [ข้อกำหนด/เงื่อนไข]...
อย่างไรก็ตาม [ข้อยกเว้น/ข้อควรระวัง]...
```

### เชื่อมการแนบเอกสาร
```
รายละเอียดปรากฏตามสิ่งที่ส่งมาด้วย
รายละเอียดตามเอกสารแนบท้ายหนังสือนี้
รายละเอียดปรากฏตามบัญชีแนบท้ายประกาศนี้
```

## สูตรสำหรับบันทึกข้อความ (ภายใน)

### ขึ้นต้น
```
ตามที่ท่านได้มอบหมายให้...
ด้วย[ฝ่าย/งาน]มีความจำเป็นต้อง...
เพื่อเป็นการ[เหตุผล]...
```

### ลงท้าย
```
จึงเรียนมาเพื่อโปรดทราบ
จึงเรียนมาเพื่อโปรดพิจารณา
จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติ
จึงเรียนมาเพื่อโปรดพิจารณาสั่งการต่อไป
```

**หมายเหตุ**: บันทึกข้อความไม่ต้องมี "ขอแสดงความนับถือ"

---

## คำพิจารณา/บันทึกของผู้บังคับบัญชา

เมื่อผู้บังคับบัญชาต้องการบันทึกความเห็น:
```
เห็นชอบ / อนุมัติ / รับทราบ
ให้ดำเนินการตามที่เสนอ
ให้ทบทวน / ให้ชี้แจงเพิ่มเติม
ไม่เห็นชอบ เนื่องจาก...
ให้[ผู้รับผิดชอบ]ดำเนินการต่อไป
```

---

## สำนวนสุภาพที่ใช้บ่อย

| สถานการณ์ | สำนวนที่ใช้ |
|----------|-----------|
| ขอโทษล่าช้า | `ขออภัยในความล่าช้า` |
| แจ้งว่าไม่สามารถทำได้ | `ขอเรียนว่า ... ไม่อยู่ในอำนาจหน้าที่ของ[หน่วยงาน]` |
| ขอให้รีบดำเนินการ | `โปรดดำเนินการโดยด่วน` |
| เน้นความสำคัญ | `ขอให้ถือเป็นเรื่องสำคัญเร่งด่วน` |
| ขอเอกสารเพิ่ม | `โปรดจัดส่งเอกสาร/หลักฐานเพิ่มเติม ดังนี้` |
| ยืนยันการรับ | `ได้รับหนังสือ/เอกสาร/พัสดุดังกล่าวแล้ว` |
| แจ้งติดตาม | `โปรดเร่งดำเนินการแจ้งผลให้ทราบโดยเร็ว` |

## สูตรสำหรับประกาศ

### ขึ้นต้น
```
โดยที่[เหตุผล/สภาพการณ์]
เพื่อให้การดำเนินงาน[ด้าน/เรื่อง]เป็นไปด้วยความเรียบร้อย
อาศัยอำนาจตามความใน[กฎหมาย/ระเบียบ] จึงประกาศ ดังนี้
```

### ลงท้ายประกาศ
```
ประกาศ ณ วันที่ [วันที่เต็ม]
ทั้งนี้ ตั้งแต่วันที่ [วันที่] เป็นต้นไป
```

---

## สูตรสำหรับคำสั่ง

### ขึ้นต้น
```
ด้วย[เหตุผล/ความจำเป็น]
เพื่อให้การปฏิบัติงาน[ด้าน/เรื่อง]เป็นไปด้วยความเรียบร้อยและมีประสิทธิภาพ
อาศัยอำนาจตามความใน[กฎหมาย/ระเบียบ] จึงมีคำสั่ง ดังนี้
```

### ลงท้ายคำสั่ง
```
ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป
ทั้งนี้ ตั้งแต่วันที่ [วันที่] เป็นต้นไป
สั่ง ณ วันที่ [วันที่เต็ม]
```
