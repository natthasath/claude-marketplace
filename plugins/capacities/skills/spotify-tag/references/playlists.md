# Playlist Reference

รายการหมวดหมู่ทั้งหมดสำหรับจับคู่เพลงกับ playlist ใน Capacities แบ่งเป็น 2 มิติ: ประเทศ และ mood/แนวเพลง

---

## มิติที่ 1: ประเทศ (Country)

ใช้เมื่อเพลงมีที่มาจากประเทศ/ภาษาที่ไม่ใช่เพลงไทยทั่วไป — ถ้าเป็นเพลงไทยปกติ **ไม่ต้องระบุประเทศ** (ถือเป็น default)

| ประเทศ | สังเกตจาก |
|---|---|
| Vietnam | เนื้อร้อง/ชื่อเพลงเวียดนาม, ศิลปินเวียดนาม, V-pop |
| China | เนื้อร้อง/ชื่อเพลงจีน, ศิลปินจีน, C-pop, เพลงประกอบซีรีส์จีน |
| Korea | เนื้อร้อง/ชื่อเพลงเกาหลี, ศิลปินเกาหลี, K-pop, OST ซีรีส์เกาหลี |
| Japan | เนื้อร้อง/ชื่อเพลงญี่ปุ่น, ศิลปินญี่ปุ่น, J-pop, City Pop, Anime OST |
| Cambodia | เนื้อร้อง/ชื่อเพลงเขมร, ศิลปินกัมพูชา |
| English | เพลงสากล, ศิลปินต่างชาติที่ร้องภาษาอังกฤษ |

---

## มิติที่ 2: Mood / แนวเพลง (Playlist)

| หมวด | Playlist อ้างอิง | ลักษณะ/บรรยากาศ |
|---|---|---|
| เพลงใต้ | ใต้ฟ้าอันดามัน, ยังมีฉันที่คิดถึงเธอ | เพลงใต้ ภาษาใต้ หรือกลิ่นอายปักษ์ใต้ |
| เพลงเหนือ | ฮักแบบคนเมือง, บ่าได้หวือหวา | เพลงเหนือ คำเมือง ล้านนา |
| เพลงอีสาน | ม่วนหลาย, อีสานขาแดนซ์ | เพลงอีสาน หมอลำ ภาษาอีสาน จังหวะม่วน |
| เพลงจีบ/ความรัก | ส่งเพลงจีบเธอทุกวัน, เธอรับรู้แต่ไม่รับรัก | จีบ, แอบชอบ, ความรักหวานๆ ยังไม่สมหวังเต็มที่ |
| เพลงอกหัก | น้ำตาไม่เคยช่วยให้ใครกลับมา แต่มันช่วยยืนยันว่าเคยรักจริง | เลิกรา อกหัก เศร้าเพราะความรัก |
| เพลงย้อนวัย | เพลงที่พาเรากลับไปในช่วงเวลาที่ดีที่สุด | เพลงเก่า ฟังแล้วคิดถึงอดีต วัยเด็ก/วัยเรียน |
| เพลงข้ามภพ | น้อนว่าง้าว, ไอต้าวกรุงศรี | แนวย้อนยุค ราชวงศ์ ตลกขบขัน มีมจากคลิป/ซีรีส์แนวข้ามภพ |
| เพลงร้านเหล้า | เกินต้าน, ในร้านเหล้า | เพลงเปิดในผับ/ร้านเหล้า จังหวะกลางๆ ดื่มเมาส์ซึ้ง |
| เพลงฮิตคาราโอเกะ | ตอนเลือกโคตรมั่นใจ ตอนร้องอยากกดข้าม | เพลงฮิตที่คนชอบเปิดคาราโอเกะ แต่ร้องยาก/คีย์สูง |
| เพลงลูกกรุง | เพลงลูกกรุง | เพลงไทยสากลยุคเก่า ทำนองหวาน คลาสสิก |
| เพลงลูกทุ่ง | เพลงลูกทุ่ง | เพลงลูกทุ่งดั้งเดิม |
| เพลงฮิตงานแต่ง | ไม่ว่าจะกี่ปีผ่านไป ก็ยังอยากแต่งกับคนเดิม | เพลงเปิดในงานแต่งงาน โรแมนติก มั่นคง |
| เพลง 3 ช่า | อีหยังวะ, ชะชะช่า | จังหวะ 3 ช่า สนุกสนาน ตลก |
| เพลงฮิต TikTok | TikTok | เพลงกำลังฮิตบน TikTok ท่อนฮุกติดหู เอาไปทำคลิป |
| เพลงแดนซ์มันส์ๆ | Dance | จังหวะเร้าใจ เต้นได้ ปาร์ตี้ |
| เพลงออกกำลังกาย (เวทเทรนนิ่ง) | Workout | จังหวะหนักแน่น มีพลัง เหมาะกับยกเวท ไม่ใช่คาร์ดิโอเบาๆ |

**หมายเหตุ:** เพลง 1 เพลงสามารถอยู่ได้หลายหมวดพร้อมกัน เช่น เพลงอีสานที่เปิดในร้านเหล้าด้วย → ทั้ง `เพลงอีสาน` และ `เพลงร้านเหล้า`
