# Mood Tag Reference

รายการ tags ทั้งหมดจัดตาม Yale Mood Meter Framework (Valence × Arousal)

---

### 🟡 Yellow — Positive + High Energy

| Tag | อารมณ์ |
|---|---|
| `#mood-energized` | มีพลังงานสูง พร้อมลุยทุกอย่าง |
| `#mood-excited` | ตื่นเต้น มีความคาดหวังสูง |
| `#mood-motivated` | มีแรงบันดาลใจ อยากลงมือทำ |
| `#mood-joyful` | ดีใจ ร่าเริง สนุกสนาน |
| `#mood-inspired` | ได้รับแรงบันดาลใจ อยากสร้างหรือแสดงออก |
| `#mood-confident` | มั่นใจในตัวเอง กล้าตัดสินใจ |
| `#mood-playful` | ขี้เล่น อารมณ์เบา ชอบตลก ไม่ซีเรียส |

### 🔴 Red — Negative + High Energy

| Tag | อารมณ์ |
|---|---|
| `#mood-stressed` | กดดัน มีภาระหนัก รู้สึกรีบ |
| `#mood-anxious` | กังวล ไม่แน่ใจในอนาคต |
| `#mood-angry` | โกรธ ไม่พอใจอย่างชัดเจน |
| `#mood-frustrated` | หัวร้อน ทำอะไรไม่ออก |
| `#mood-overwhelmed` | รับไม่ไหว มากเกินไป |
| `#mood-irritated` | หงุดหงิด เซ็ง ถูกรบกวน |
| `#mood-defensive` | ตั้งรับ รู้สึกถูกโจมตี |

### 🟢 Green — Positive + Low Energy

| Tag | อารมณ์ |
|---|---|
| `#mood-calm` | สงบ ผ่อนคลาย ไม่มีความกังวล |
| `#mood-content` | พึงพอใจในสิ่งที่มี รู้สึกพอเพียง |
| `#mood-grateful` | ขอบคุณ ซาบซึ้งใจ |
| `#mood-focused` | จดจ่อกับงาน จิตใจไม่วอกแวก |
| `#mood-hopeful` | มีความหวัง เชื่อว่าจะดีขึ้น |
| `#mood-at-ease` | สบายใจ ไม่มีอะไรกวนใจ |
| `#mood-reflective` | ใคร่ครวญ ทบทวนตัวเอง |

### 🔵 Blue — Negative + Low Energy

| Tag | อารมณ์ |
|---|---|
| `#mood-sad` | เศร้า หดหู่ รู้สึกหนักใจ |
| `#mood-lonely` | เดียวดาย ต้องการการเชื่อมต่อ |
| `#mood-drained` | หมดแรง อ่อนล้าทางอารมณ์ |
| `#mood-disappointed` | ผิดหวัง สิ่งที่คาดหวังไม่เป็นจริง |
| `#mood-unmotivated` | ไม่อยากทำอะไร ขาดแรงขับ |
| `#mood-unsettled` | รู้สึกไม่ลงตัว บอกไม่ถูก มีบางอย่างค้างคาใจ |
| `#mood-numb` | ชา ไม่รู้สึกอะไร อารมณ์ราบเรียบผิดปกติ |
