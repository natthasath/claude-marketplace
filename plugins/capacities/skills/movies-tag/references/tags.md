# Genre Tag Reference

รายการ tags ทั้งหมดพร้อมคำอธิบายภาษาไทย จัดกลุ่มตามประเภท

---

## ตัวละคร / Archetype

| Tag | ความหมาย |
|---|---|
| `#genre-spy` | สายลับ ภารกิจลับ ปลอมตัว |
| `#genre-superhero` | ซูเปอร์ฮีโร่ พลังพิเศษ |
| `#genre-samurai` | ซามูไร นักดาบ บูชิโด |
| `#genre-profiler` | นักอ่านพฤติกรรม FBI profiler สายสืบจิตวิทยา |
| `#genre-genius` | อัจฉริยะ คนฉลาดเกินมนุษย์ |
| `#genre-champion` | แชมป์ นักสู้ ต่อสู้เพื่อเป็นที่หนึ่ง |
| `#genre-detective` | นักสืบ สืบคดี หาหลักฐาน |

## อาชญากรรม / Crime

| Tag | ความหมาย |
|---|---|
| `#genre-crime` | อาชญากรรม โลกใต้ดิน |
| `#genre-heist` | ปล้น วางแผนปล้น Ocean's Eleven style |
| `#genre-murder` | ฆาตกรรม ศพ คนตาย |
| `#genre-hijack` | จี้ ยึด จับตัวประกัน |
| `#genre-prison` | ในคุก เรือนจำ แหกคุก |
| `#genre-darkside` | โลกมืด มุมมืดของมนุษย์ ตัวร้ายที่เข้าใจได้ |
| `#genre-revenge` | แก้แค้น ล้างแค้น |

|---|---|
| `#genre-mystery` | ลึกลับ ปริศนา ไม่รู้ว่าใครทำ |
