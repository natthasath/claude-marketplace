# ตัวอย่างผลลัพธ์ที่ถูกต้อง (Few-shot Examples)

ใช้ตัวอย่างเหล่านี้เป็น reference สำหรับ tone, ความยาว และสไตล์การเขียนที่ถูกต้อง

---

## หัวข้ออบรม: Cyber Security

**ความรู้ที่ได้รับจากการพัฒนาตนเอง**
ได้เรียนรู้ว่า การรักษาความมั่นคงปลอดภัยเริ่มจากความเข้าใจระบบ เพราะคนที่ไม่เข้าใจความเสี่ยง มักเป็นผู้สร้างช่องโหว่ได้เร็วกว่า Hacker เสมอ

**การนำความรู้จากการพัฒนาตนเองมาประยุกต์ใช้ในการปฏิบัติงาน**
ประยุกต์ใช้ความรู้โดยเน้นการเฝ้าระวัง วิเคราะห์ และป้องกันเชิงรุกอย่างต่อเนื่อง เพราะการทำงานด้าน Cyber Security ไม่ควรวัดจากจำนวนเหตุการณ์ที่แก้ไขได้ แต่ควรวัดจากจำนวนเหตุการณ์ที่ป้องกันไม่ให้เกิดขึ้นได้ตั้งแต่แรก

**การประเมินผลสัมฤทธิ์จากการพัฒนาตนเอง (ตอบโดยผู้บังคับบัญชาของผู้เข้ารับการอบรม)**
สังเกตว่าผู้เข้าอบรมเริ่มตั้งคำถามเรื่อง access control และ logging ก่อน deploy ไม่ใช่หลัง incident เกิดขึ้นแล้ว เช่นเดียวกับที่ Cyber Security สอนว่าช่องโหว่ที่อันตรายที่สุดคือช่องโหว่ที่ไม่มีใครคิดว่าจะโดน — วัดผลไม่ใช่จากจำนวนครั้งที่แก้ไข
ปัญหาได้ แต่จากจำนวนครั้งที่ป้องกันปัญหาก่อนที่มันจะเกิด
